CFGMS

Modern, secure configuration management system designed for resilience, security, and clean architecture

View on GitHub Documentation

Key Features

🔒 Zero-Trust Security

Mutual TLS authentication, continuous authorization, and cryptographic configuration signing for defense-in-depth protection.

🏢 Multi-Tenant Architecture

Hierarchical tenant model (MSP → Client → Group → Device) designed to scale to 50k+ endpoints.

⚡ Hybrid Communication

MQTT control plane for real-time commands with QUIC data plane for high-performance synchronization.

🔌 Pluggable Architecture

Modular storage, logging, secrets management, and directory services with clean interfaces.

☁️ Microsoft 365 Integration

Native Entra ID, Conditional Access, and Microsoft 365 management for MSP deployments.

🎯 Declarative Configuration

GitOps-ready with encrypted secret storage (SOPS) and full audit trails for compliance.

Get Started
