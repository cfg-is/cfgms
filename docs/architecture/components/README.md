## Overview

CFGMS is built around a set of core components that work together to provide configuration management capabilities:

1. **Controller**: The central management server
2. **Steward**: The cross-platform agent that runs on managed endpoints
3. **Outpost**: The proxy cache agent for network environments
4. **Specialized Stewards**: Components that extend core functionality to specific environments
