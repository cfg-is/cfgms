# Development Guidelines

This directory contains documentation related to development standards, guidelines, and processes for CFGMS.

## Overview

The development documentation provides guidance for developers working on CFGMS, including:

- Coding standards and best practices
- Development workflow and processes
- Testing requirements and methodologies
- Documentation standards
- AI integration guidelines
- Getting started guides for new developers

## Directory Structure

- **guides/**: Contains guides for developers, including getting started guides, workflow guides, and best practices.
- **standards/**: Contains development standards, including coding standards, testing standards, and documentation standards.

## Key Documents

- **Getting Started Guide**: Introduction to CFGMS development for new developers
- **Coding Standards**: Standards for Go code in CFGMS
- **Testing Standards**: Standards for testing in CFGMS
- **Documentation Standards**: Standards for documentation in CFGMS
- **Development Workflow**: Overview of the development workflow in CFGMS
- **AI Integration Guidelines**: Guidelines for integrating AI tools into the development process
